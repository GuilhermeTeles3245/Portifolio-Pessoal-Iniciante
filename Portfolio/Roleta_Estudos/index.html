🎲 Roleta dos Estudos

Crie uma lista com os cursos que você está estudando. A roleta vai sortear a próxima aula de forma aleatória, mantendo a ordem crescente dentro de cada curso.

Exemplo de lista:

  • 📘 HTML (10 aulas)
  • 📗 CSS (8 aulas)
  • 📙 JavaScript (12 aulas)

A ordem sorteada pode ser algo como:

1º JavaScript - 1 2º HTML - 1 3º CSS - 1
🔗 Abrir outra página
⏳ Clique no botão para sortear
